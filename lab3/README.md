# Аудиоплеер Aphex Twin

Плеер с кастомным интерфейсом. Окно без рамок, прозрачный фон, изображение головы Aphex Twin.

## Управление окном

Окно без стандартных кнопок. Перетаскивается мышью. Есть кнопки закрытия и сворачивания. Масштаб от 10% до 200% через меню.

## Воспроизведение

Поддерживает MP3, WAV, OGG, FLAC, M4A, AAC. Кнопки play/pause и stop. Прогресс-бар с перемоткой по клику или перетаскиванием. Показывает текущее время и длительность. Строка состояния с названием трека и автором, бегущая строка если текст не помещается.

## Визуализация

Три режима, переключаются кнопками на интерфейсе или клавишами 1/2/3:
- **Волна** (клавиша 1) — осциллограф по волновой форме
- **3D спектр** (клавиша 2) — кольцевой эффект
- **2D кот** (клавиша 3) — gif-анимация, играет только при воспроизведении

Визуализатор использует FFT для анализа спектра и автогейн для нормализации амплитуды.

## Горячие клавиши

- `Space` — play/pause
- `S` — стоп
- `1` — режим волны
- `2` — 3D спектр
- `3` — 2D кот
- `Ctrl+O` — открыть файл
- `Ctrl+Q` — выход

## Меню

На macOS меню в системной строке сверху. На Windows и Linux — контекстное меню по правой кнопке мыши.

Разделы: Файл, Воспроизведение, Визуализация, Масштаб, Выход.


## Технологии

- PyQt6 — интерфейс
- PyQt6-Multimedia — воспроизведение аудио
- NumPy — обработка сигнала для визуализации
- Pillow — работа с изображениями

## Платформы

Работает на macOS, Windows, Linux. На Windows и Linux меню вызывается правой кнопкой мыши.

## Сборка

Для создания .app, .exe или .dmg смотри инструкции в `build/README.md`.

Быстрый старт:
```bash
# macOS
cd build && ./build_macos.sh

# Windows
cd build && build_windows.bat

# Linux
cd build && pyinstaller build.spec
```
