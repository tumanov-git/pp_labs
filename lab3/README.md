# Аудиоплеер на PyQt6

Каркас аудиоплеера с кастомным интерфейсом в виде головы Aphex Twin.

## Структура проекта

```
lab3/
├── player/              # Основной модуль плеера
│   ├── __init__.py     # Инициализация модуля
│   ├── main.py         # Точка входа в приложение
│   ├── window.py       # Главное окно с UI
│   ├── audio_player.py # Класс для воспроизведения аудио
│   └── config.py       # Конфигурация (пути к файлам)
├── UI/                 # UI ресурсы
│   ├── aphex.png       # Изображение головы Aphex Twin
│   ├── close.png       # Кнопка закрытия
│   └── minimize.png    # Кнопка сворачивания
├── run_player.py       # Скрипт для запуска
└── requirements.txt    # Зависимости проекта
```

## Установка

1. Установите зависимости:
```bash
pip install -r requirements.txt
```

## Запуск

```bash
python run_player.py
```

или

```bash
python -m player.main
```

## Функциональность

### Реализовано:
- ✅ Прозрачное окно без стандартного интерфейса macOS
- ✅ Фон с изображением головы Aphex Twin
- ✅ Кнопки закрытия и сворачивания на координатах X: 400, Y: 15
- ✅ Перетаскивание окна мышью
- ✅ Базовый каркас для воспроизведения аудио (MP3, WAV, OGG и др.)

### В разработке:
- ⏳ Кнопки управления воспроизведением (play, pause, stop)
- ⏳ Ползунок прогресса воспроизведения

## Технические детали

- **PyQt6** - для создания GUI
- **PyQt6-Multimedia** - для воспроизведения аудио
- Окно имеет фиксированный размер, равный размеру изображения aphex.png
- Кнопки расположены поверх изображения с прозрачным фоном
- Расстояние между кнопками: 5 пикселей
