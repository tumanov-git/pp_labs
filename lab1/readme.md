## Лор болотного курорта «У Шрека»

Болотный курорт — это небольшой природный оздоровительный комплекс, расположенный на территории болотной зоны. Курорт предлагает услуги отдыха и восстановления с использованием природных ресурсов: грязевых ванн, банных процедур, массажей, прогулок по территории и индивидуальных оздоровительных программ.
